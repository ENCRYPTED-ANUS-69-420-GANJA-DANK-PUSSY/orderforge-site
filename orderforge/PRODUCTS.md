# Initial Product Plan

## 1. Desk Reset Kit
Target: desk workers, remote workers, gamers, students
Price: $7

Contents:
- Desk layout planner
- Cable label sheets
- Device inventory sheet
- Workspace reset checklist
- Drawer category planner
- Daily/weekly desk reset sheet

Why it sells:
- Broad audience
- Easy to understand
- Immediate practical use
- Strong bridge into future physical products

## 2. Workshop Starter Pack
Target: makers, DIY users, garage/workbench owners
Price: $9

Contents:
- Tool inventory sheets
- Hardware bin labels
- Pegboard planning template
- Maintenance log
- Storage mapping sheets

## 3. Small Space Storage Bundle
Target: apartment dwellers, dorms, small rooms
Price: $7

Contents:
- Bin labels
- Drawer labels
- Shelf zoning sheets
- Declutter checklists
- Storage category planner

## 4. Gaming Setup Organizer Pack
Target: PC gamers, streamers, console users
Price: $7

Contents:
- Setup planner
- Cable map sheet
- Accessory inventory sheet
- Reset checklist
- Zone labels

## 5. Editable Label Pack
Target: broad audience
Price: $5

Contents:
- Printable labels
- Editable label templates
- Category tags for bins, drawers, tools, cables, accessories

## Future physical product mapping
Desk Reset Kit -> cable clips, under-desk hooks, drawer organizers, label holders
Workshop Starter Pack -> tool holders, battery holders, pegboard accessories
Gaming Setup Organizer Pack -> controller stands, headset hooks, cable clips
Small Space Bundle -> compact organizers, drawer dividers, wall hooks
Editable Label Pack -> clip-in label frames, bin tags, modular signage
