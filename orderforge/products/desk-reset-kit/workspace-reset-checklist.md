# Workspace Reset Checklist

## Daily reset
- Put loose items back
- Return tools/accessories to their zones
- Clear dishes/trash
- Coil or tuck stray cables
- Put chargers back in place
- Wipe key surfaces

## Weekly reset
- Review what is piling up
- Remove items that do not belong
- Re-label any confusing storage
- Re-route problem cables
- Check charging station clutter
- Clean dust from main surfaces
